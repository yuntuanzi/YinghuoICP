## 🎀 项目介绍

**YinghuoICP（萤火ICP）** 是基于 [TuanICP](https://github.com/yuntuanzi/TuanICP/) 精心打造的🎨**新二次元风格**虚拟备案系统。在保留原有强大功能的基础上，为您带来全新的动漫视觉体验！

- 🎯 **一键切换** - 原有TuanICP用户可直接体验新主题
- 🔒 **安全备案** - 完善的虚拟备案管理机制
- 📱 **多端适配** - 完美支持PC、平板、手机访问
- 🚀 **极速体验** - 优化的代码结构

## 🛠 安装指南

### 📋 环境要求

| 组件 | 版本要求 | 备注 |
|------|----------|------|
| PHP | ≥ 8.1 | 核心运行环境 |
| MySQL | ≥ 5.7 | 数据库支持 |
| Redis | 最新版 | 缓存服务 |
| Web服务器 | Nginx/Apache | 网站服务 |

### 🔧 安装步骤

#### 1️⃣ 下载源码
可直接下载ZIP包解压

#### 2️⃣ 环境配置
确保安装以下PHP扩展：
- 🎯 `fileinfo` - 文件处理
- 🎯 `redis` - 缓存服务

#### 3️⃣ 服务器设置
- 🎯 站点根目录设置为：`public`
- 🎯 放行 **465端口**（邮件服务）

#### 4️⃣ 安装系统
访问安装页面开始配置：
```
🌐 http(s)://您的域名/install.php
```

#### 5️⃣ 访问后台
安装完成后，通过以下地址管理您的系统：
```
🔧 http(s)://您的域名/admin
```

## 🎯 兼容说明

> 💖 **重要提示**：YinghuoICP 是 TuanICP 的**主题版本**，数据库结构完全一致！

### 🔄 无缝迁移
- ✅ **数据库通用** - 直接使用原有TuanICP数据库
- ✅ **数据保留** - 用户、备案数据完整继承
- ✅ **零配置** - 安装即用，无需额外设置

### 🎨 主题优势
- 在享受全新二次元界面的同时，保留所有熟悉的功能
